# Welcome to the Open Source Voice Agent Platform

This platform provides a complete, scalable infrastructure for building, deploying, and managing real-time voice AI agents. It integrates **LiveKit** for low-latency WebRTC processing and **Vobiz** for reliable SIP telephony.

## üõ†Ô∏è Architecture

The platform is built on a modern microservices architecture:

- **Gateway Service**: Unified API entry point.
- **Config Service**: Management of assistants, tools, and phone numbers.
- **Orchestration Service**: Handling of campaigns and tasks.
- **Analytics Service**: Call logs, analysis, and storage.
- **Agent Worker**: The brain of the AI, running LiveKit agent processes.
